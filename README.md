# 🌌 PHYSICS GALAXY - 超絶リアルな3D物理シミュレーション

## 概要
本格的な物理エンジンを搭載した3D物理シミュレーター。ニュートン力学から流体力学まで、様々な物理現象をリアルタイムで体験できる教育・研究ツールです。

## 🚀 主要機能

### ⚛️ 物理エンジン
- **重力シミュレーション**: 可変重力場での落下実験
- **衝突判定**: 弾性・非弾性衝突の完全シミュレーション
- **摩擦・反発**: リアルな表面相互作用
- **空気抵抗**: 流体力学に基づく抵抗計算
- **バネ振動**: フックの法則による弾性力
- **万有引力**: 多体系での重力相互作用

### 🎮 インタラクション
- **リアルタイム作成**: クリックでオブジェクト生成
- **力の適用**: ドラッグで力ベクトル入力
- **オブジェクト選択**: 詳細な物理情報表示
- **3Dカメラ**: ズーム・回転による視点変更
- **パラメータ調整**: リアルタイム物理定数変更

### 📊 可視化システム
- **軌跡表示**: オブジェクトの運動経路
- **速度ベクトル**: 瞬間速度の方向と大きさ
- **力ベクトル**: 作用する力の可視化
- **エネルギー表示**: 運動・位置エネルギーの監視
- **統計情報**: FPS、オブジェクト数、総エネルギー

## 🎯 プリセットシナリオ

### 🍎 重力実験
ガリレイの落下法則を検証。異なる質量の物体が同時に落下する様子を観察。

**物理法則**: F = mg

### ⚖️ 振り子実験
単振り子の周期運動を解析。エネルギー保存則と調和振動の基本を学習。

**物理法則**: T = 2π√(L/g)

### 💥 衝突実験
弾性・非弾性衝突による運動量保存則の実証。

**物理法則**: m₁v₁ + m₂v₂ = m₁v₁' + m₂v₂'

### 🌍 軌道運動
万有引力による惑星運動のシミュレーション。ケプラーの法則を体験。

**物理法則**: F = G(m₁m₂)/r²

### 🌀 バネ振動
フックの法則による単純調和運動。減衰振動とエネルギー散逸。

**物理法則**: F = -kx

### 🌊 流体力学
粘性流体中での物体運動。終端速度と浮力の効果。

**物理法則**: F = ½ρv²CdA
